## Project Context
This project analyzes a dataset of vehicle listings to understand what factors influence vehicle prices.

## Analysis
The project involved cleaning the dataset, exploring relationships between variables such as mileage, vehicle age, and condition, and visualizing price patterns using Python.

Tools used:
- Python
- Pandas
- Matplotlib
- Exploratory Data Analysis

## Conclusions
The analysis showed that vehicle age and mileage have a strong influence on price. Vehicles with lower mileage and newer models tend to maintain higher values in the market.
